NC News API

The NC News API provides endpoints to access, filter, and manage news articles, topics, users, and comments. It supports sorting, ordering, and topic filtering, and is designed for both development and production use.

Available Endpoints:

  • GET /api/topics

    Retrieve a list of all topics.

    Example response:

                
{
  "topics":
  [
    {
      "slug": "coding",
      "description": "Code is love, code is life"
    },
    {
      "slug": "football",
      "description": "FOOTIE!"
    },
    {
      "slug": "cooking",
      "description": "Hey good looking, what you got cooking?"
    }
  ]
}

  • GET /api/articles

    Retrieve all articles. Supports sorting, ordering, and filtering by topic.

    Query Parameters (optional):

    • sort_by - Sort by any valid column (e.g., created_at, votes, title). Default: created_at
    • order - asc or desc. Default: desc
    • topic - Filter articles by topic slug

    Example response:

                
{
  "articles": [
    {
      "article_id": 1,
      "title": "Living in the shadow of a great man",
      "topic": "mitch",
      "author": "butter_bridge",
      "created_at": "2020-07-09T20:11:00.000Z",
      "votes": 100,
      "article_img_url": "https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700",
      "comment_count": 11
    }
  ]
}

  • GET /api/articles/:article_id

    Retrieve a single article by its ID.

    Example response:

                
{
  "article": {
    "article_id": 1,
    "title": "Living in the shadow of a great man",
    "topic": "mitch",
    "author": "butter_bridge",
    "body": "I find this existence challenging",
    "created_at": "2020-07-09T20:11:00.000Z",
    "votes": 100,
    "article_img_url": "https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700"
  }
}

  • GET /api/articles/:article_id/comments

    Retrieve all comments for a specific article.

    Example response:

                
{
  "comments": [
    {
      "comment_id": 5,
      "body": "I hate streaming noses",
      "article_id": 1,
      "author": "icellusedkars",
      "votes": 0,
      "created_at": "2020-11-03T21:00:00.000Z"
    }
  ]
}

  • POST /api/articles/:article_id/comments

    Add a new comment to an article.

    Request body:

                
{
  "username": "butter_bridge",
  "body": "This is a test comment"
}

    Example response:

                
{
  "comment": {
    "comment_id": 19,
    "body": "This is a test comment",
    "article_id": 1,
    "author": "butter_bridge",
    "votes": 0,
    "created_at": "2024-01-10T12:00:00.000Z"
  }
}

  • PATCH /api/articles/:article_id

    Update the vote count for an article.

    Request body:

                
{
  "inc_votes": 1
}

    Example response:

                
{
  "article": {
    "article_id": 1,
    "title": "Living in the shadow of a great man",
    "topic": "mitch",
    "author": "butter_bridge",
    "body": "I find this existence challenging",
    "created_at": "2020-07-09T20:11:00.000Z",
    "votes": 101,
    "article_img_url": "https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700"
  }
}

  • DELETE /api/comments/:comment_id

    Delete a comment by its ID. Returns no content on success.

    Example response:

                
Status: 204 No Content

  • GET /api/users

    Retrieve a list of all users.

    Example response:

                
{
  "users": [
    {
      "username": "butter_bridge",
      "name": "jonny",
      "avatar_url": "https://www.healthytherapies.com/wp-content/uploads/2016/06/Lime3.jpg"
    }
  ]
}
